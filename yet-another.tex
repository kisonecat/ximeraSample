We are calculating the value of $\displaystyle \int_{-1}^{2} f(x)  x$.
 

We start by setting up the Riemann sum using $n$ rectangles using right endpoints.
What is $\Delta x$?
     
\[ \Delta x = \answer{3/n} \]
